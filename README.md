This project is an AI-powered Fake News Detection Web Application that classifies news text as:

1. Real News
2. Fake News

"It uses Machine Learning (NLP) with a React frontend and Flask backend."

 **TECH STACK ARE USED**
> Frontend
 1.React (JSX)
 2.Fetch API
> Backend
1.Python (Flask)
2.Flask-CORS
> Machine Learning
1.Scikit-learn
2.TF-IDF Vectorizer
3.Logistic Regression
